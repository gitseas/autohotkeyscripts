# Windows Productivity Scripts
A collection of AHK v2 scripts for window management and app launching.
## Hotkeys
| Key                | Action | File |
| :----------------: | :---------------------: | :-------------: |
| **Win + B**        | Launch LibreWolf        | master.ahk      |
| **Win + N**        | Launch Obsidian         | master.ahk      |
| **Win + Enter**    | Launch WSL              | master.ahk      |
| **Win + M**        | Toggle Maximize/Restore | master.ahk      |
| **Win + F**        | Center Window           | master.ahk      |
| **Ctrl + Alt + R** | Reload Master           | master.ahk      |
| **Win + W**        | Close Window (Admin)    | admin_tools.ahk |
